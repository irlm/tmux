# Features and Project Layout

## What This Repo Owns

This repository is the tmux and terminal side of the environment. It includes:

- tmux configuration in `tmux.conf`
- installer scripts for macOS, Linux, and Windows
- helper scripts for popups, search, notes, prompts, and status bar segments
- shell-oriented quality-of-life setup

The Neovim configuration lives in a separate repo: [irlm/nvim](https://github.com/irlm/nvim).

## Main Features

### Tmux Workflow

- `C-a` prefix instead of `C-b`
- vim-style pane movement with `h j k l`
- popup-driven workflow for `lazygit`, `lazydocker`, system info, notes, and search
- session and window helpers for switching, renaming, detaching, and remote work

### Neovim Integration

- designed to pair with the separate LazyVim-based Neovim config
- tmux/Neovim navigation through `vim-tmux-navigator`
- installer scripts clone both repos together
- markdown preview: inline rendering (`Space mr`) and live browser preview (`Space mp`)

### Status Bar

The right side of the tmux status bar is generated by `scripts/status_right.sh`. It pulls together:

- network speed
- CPU usage
- RAM usage
- battery status when available
- current time

The left side shows the session name, current git branch when present, and sync state. In nested tmux mode it temporarily switches to a visible `REMOTE` indicator.

### Session Persistence

The config includes:

- `tmux-resurrect` for save and restore
- `tmux-continuum` for periodic autosave and optional restore

Pane contents are also captured for restore support.

### Search and Reference Helpers

The repo includes popup helpers for:

- web search
- quick AI questions
- `cheat.sh`
- `tldr`
- man pages
