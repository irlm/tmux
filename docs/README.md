# Documentation

This folder contains the full documentation for the tmux environment.

The root [`README.md`](../README.md) is the quick-start version. Use the guides here when you want the full picture.

## Start Here

- [Installation guide](installation.md): install modes, platform notes, first-run checklist, and updating
- [Usage and keybindings](usage.md): the main tmux workflow, popups, pane/session management, copy mode, and shell aliases
- [Features and project layout](features.md): what the project includes, how the status bar and plugins fit together, and which files do what

## Project Summary

This repo owns the tmux configuration, shell integration, helper scripts, and installer scripts. It pairs with the separate [irlm/nvim](https://github.com/irlm/nvim) repo for the Neovim configuration.

If you are new to the project, the best path is:

1. Read the root [`README.md`](../README.md)
2. Follow the [installation guide](installation.md)
3. Keep the [usage guide](usage.md) open for the first few sessions
