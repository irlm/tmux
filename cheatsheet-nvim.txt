BASICS (vim)               MOVEMENT                    CLOSE / QUIT
 i    insert mode           h j k l   char/line        :q     close window
 a    append after cursor   w / b     word fwd/back    :bd    close buffer
 A    append end of line    e         end of word      Spc bd  close buffer
 o    new line below        0 / $     start/end line   Spc wd  close window
 O    new line above        gg / G    top/bottom file  Spc qq  quit ALL nvim
 v    visual select         { / }     paragraph        Esc     back to normal
 V    visual line           C-u / C-d half page up/dn  q       close popup/list
 C-v  visual block          %         matching bracket
 .    repeat last change    f/F x     jump to char x
────────────────────────────────────────────────────────────────────────────────
SELECT / EDIT              COPY & PASTE                UNDO / REDO
 viw  select word           y     yank (copy)          u     undo
 vi"  select in quotes      yy    yank line            C-r   redo
 vib  select in parens      d     delete (cut)
 viB  select in braces      dd    delete line         SEARCH IN FILE
 vat  select around tag     p     paste after          /     search forward
 ggVG select ALL            P     paste before         ?     search backward
 >    indent selection      "+y   yank to clipboard    n / N next/prev match
 <    unindent selection    "+p   paste from clipboard  *     search word under cursor
 gco  comment below         x     delete char          Spc ur clear highlights
 gcO  comment above
 gc   toggle comment (vis)

FILES & BUFFERS            SEARCH (picker)             WINDOWS

 Spc fr recent files        Spc sG  grep (cwd)         Spc wd  close window
 Spc fb buffers             Spc sw  grep word/sel      Spc wm  maximize toggle
 Spc fn new file            Spc sf  find files         C-h/j/k/l navigate panes
 Spc ,  switch buffer       Spc sb  buffer lines       C-Up/Dn resize
 S-h   prev buffer          Spc sk  keymaps            C-Left/Right resize
 S-l   next buffer          Spc sh  help pages
 Spc bb other buffer        Spc sc  command history
 Spc bo close other bufs    Spc s"  registers
 C-s   save file            Spc :   command history
